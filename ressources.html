Ressources

Durant la réalisation de notre projet nous avons utilisé divers sites afin de nous faciliter la tâche. Nous avons décidé de vous les partager ainsi que de vous expliquer en quoi ils nous ont servi.

JetBrains Rider

Rider est un environnement très efficace qui nous permet de programmer sous csharp qui est le langage principale que nous utilisons .

Unity

Nous avons décidé d'utiliser le moteur de jeu Unity 3D car il possède plusieurs features très bien pensées, et il est relativement simple à prendre en main. Il nous permettra notamment d'implémenter le mode de jeu avec l'IA.

Blender / GIMP

Pour les visuels nous avons fait appel à Blender pour la partie 3D ainsi que GIMP pour celle de la 2D.

Photon Engine

Photon nous a été d'une grand aide pour implémenter notre mode Multijoueur.

Bootstrap Studio 5

Afin de réaliser le site web rapidement, sans utilisation de code HTML , Bootstrap est un outil très efficace simple a utilisé.

FL Studio 20

FL Studio 20 a été utilisé pour composer l'OST (Original Sound Track) dans un style Phonk.

LATEX

Nous utilisons LATEX pour effectuer nos rapports et notre cahier des charges. 

Assets

Ces deux assets nommé respectivement POLYGON Street Race et POLYGON City Pack nous ont été d'une très grand aide pour la réalisation de divers designs et mécanismes
