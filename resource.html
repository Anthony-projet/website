Resources

During our project, we use some websites and softwares to help us. They are here.

JetBrains Rider

Rider  is a IDE very efficient and we use them to develop in CSharp.

Unity3D

We decided to use the Unity 3D game engine because it has several very well thought out features, and it is relatively easy to learn. It will allow us to implement the AI game mode.

Blender / GIMP

For the visuals we used Blender for the 3D part and GIMP for the 2D part.

Photon Engine
Photon Unity Networking

Photon is our solution of networking to develop the multiplayer mode.

Bootstrap Studio 5

To create the website, we use Bootstrap Studio

FL Studio 20

To compose our OST (Original SoundTrack), we use FL Studio.

LaTeX

To create our reports, and prepare our defenses, we use LaTeX via the website named Overleaf.

Assets

We use these two assets to create our tracks, and create our cars.
